# Button

A button. Pretty self explantory



## Importing

```jsx
import { Button } from "@vizality/components"
```



## Example

```jsx
<Button
    color={Button.Colors.BRAND}
    look={Button.Looks.FILLED}
    size={Button.Sizes.MEDIUM}
>
	bruh
</Button>
```
